# Howdy from [Leptos]()


## Prerequisites

```bash
brew update
brew install pkg-config openssl
brew install rustup

# add the path to your .zshrc file:
echo 'export PATH="$(brew --prefix rustup)/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc

# confirm rust and cargo are operational
rustup default stable
rustc --version
cargo --version

cargo install cargo-watch
```
