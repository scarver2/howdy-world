# Howdy from [InertiaJS](https://inertiajs.com/)

## Run Locally

```bash
docker compose up --build
open http://localhost:3000/javascript-inertia/

```
or

```bash
npm run dev:ts
```

and visit http://localhost:3000/

## Build and Run

```bash
npm run typecheck
npm run build
node dist/server.js
```


```
  package.json
  server.ts
  public/
    app.js
    Pages/
      Home.js
```
