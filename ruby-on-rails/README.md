# Ruby on Rails

## Notes

A Rails application with the absolute minimum number of files was generated in order to demonstrate deployment functionality.

The following command was used to generate the application:

```bash
rails new Howdy --minimal  --skip-active-record  --skip-test  --skip-git  --skip-javascript  --skip-action-mailer  --skip-active-storage  --skip-hotwire  --skip-sprockets  --skip-action-mailbox --skip-action-text --skip-active-job --skip-action-cable --skip-asset-pipeline --skip-jbuilder --skip-dev-gems --skip-decrypted-diffs
```
