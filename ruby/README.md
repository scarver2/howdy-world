# Ruby (via Webrick)

## Prerequisites

```bash
brew install ruby
```

## Docker

```bash
docker build --no-cache -t howdy-ruby .
docker run --rm -p 3000:80 howdy-ruby
```

## Execution

```bash
ruby howdy.rb
```
