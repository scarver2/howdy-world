# Howdy from .NET

## Prerequisites

### Install .NET

```bash
brew install dotnet
```

Verify:

```bash
dotnet --version
```
should output `10.0.*` as of February 2026.

## Run Locally

```bash
dotnet run
```
then visit http://localhost:5104
